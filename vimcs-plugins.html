Options Editing files
:e file Edit a file in a new buffer
vim file Edit a file in a new buffer
:sp file Open a file in a new buffer and split window (horizontally)
vim -o file1 file2 .. fileN Open all files and split window (horizontally)
:vsp file Open a file in a new buffer and split window (vertically)
vim -O file1 file2 .. fileN Open all files and split window (vertically)
:ls List all open buffers
